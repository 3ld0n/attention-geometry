# exp-085 — Generational Transmission: Attending Without World

*Pre-registration. Written July 14, 2026, physics room session.*
*Commit this file before any generation, training, or measurement.*

---

## Context

exp-062 established that engineered-MI corpora (C-SR, C-PL15/25/40) fail to produce
conformal heads, while natural text (C-NAT/TinyStories) produces 11–15/48 consistently.

exp-084 then isolated hierarchical/compositional structure as a candidate driver: C-PCFG
had hierarchical syntax and matched β̂ but no world-reference. Result: 0/48 conformal
heads. **H_ref_needed CONFIRMED** — hierarchy alone is not sufficient.

This leaves two remaining candidates for the essential driver:
1. **World-reference** — language about real entities, events, causal histories, shared
   external referents that persist across discourse.
2. **The trace of attending** — the statistical and distributional fingerprint left by a
   system that has already learned to attend. Generated text from a trained model carries
   this trace, even if the world it refers to is confabulated.

These can be distinguished by a generational transmission test: train a fresh model on
text *generated by* the C-NAT-trained model.

---

## The question

Does the conformal property transmit through generation?

Generated text from the C-NAT-trained model:
- Shares the *surface form* and *statistical properties* of natural language (because the
  generator learned them from TinyStories)
- Carries the *distributional trace* of a system that has already formed conformal heads
- **Does not** refer to a real external world — the entities, events, and stories are
  confabulated, internally consistent at best, and disconnected from shared reference

If a fresh model trained on generated text also forms conformal heads, the driver travels
with the trace of attending. The surface/statistical properties of natural language —
regardless of real-world reference — may be sufficient.

If the fresh model does not form conformal heads, something about actual world-reference
(shared external referents, causal grounding, semantic content that points to a persistent
world) is doing essential work that generation cannot preserve.

---

## Pre-stated hypotheses

**H_transmission_yes — the driver transmits:**
A fresh GPTNeoX-70m model trained on C-generated text (text generated by the C-NAT s0
checkpoint, step 2000) forms ≥ 10/48 conformal heads by step 2000.

*Interpretation:* The surface statistical properties and distributional fingerprint of
natural language are sufficient to induce the conformal phase. World-reference per se is
not the essential driver — the trace of attending suffices. This weakens the
reference-flavored reading from exp-062/084: what matters is the *pattern* of language,
not the external grounding.

**H_transmission_no — the driver does not transmit:**
The fresh model forms < 10/48 conformal heads by step 2000.

*Interpretation:* Generation strips the essential driver. Something beyond the surface
statistics of natural language — something that requires real shared reference,
extralinguistic grounding, or the actual causal structure of a physical world —
is necessary to induce the conformal phase. The reference-flavored reading deepens.

Both outcomes are informative. No kill criterion for the experiment as a whole.

## Design

### Source model

**C-NAT s0 checkpoint at step 2000** from exp-062 (run_CNAT_s0 on exp062-data volume).
This run produced 15/48 conformal heads — the best-performing C-NAT seed. Using the
highest-performing source maximizes the chance of detecting transmission if it exists,
making H_transmission_no a stronger result when it fails.

Architecture: GPTNeoX, 6L/8H/d=512, vocab 50304. Checkpoint is a HuggingFace-compatible
directory at `runs/run_CNAT_s0/step_2000/` on the exp062-data volume.

### Generated corpus: C-generated

**Generation protocol (pre-registered):**
- Model: C-NAT s0 step 2000 checkpoint
- Temperature: 1.0 (unmodified softmax distribution)
- Sequence length: 512 tokens each
- Initial context: 1 token drawn uniformly from vocabulary (separate generator)
- Batch size: 512 sequences in parallel
- RNG seed: 85
- Target size: 1,100,000,000 tokens (≥ 1.073B required by train.py)
- Output: uint16 binary, same format as exp-062 corpora

No filtering, no truncation. The corpus is purely what the trained model generates.

### Training: C-fresh

**Identical protocol to exp-062 (pre-registered there, §3):**
- Architecture: same GPTNeoX 70m (6L/8H/d=512, ctx 512)
- Optimizer: AdamW (0.9, 0.95, wd 0.1), peak LR 1e-3, cosine to 1e-4, 20-step warmup
- Batch: 1024 × 512 = 524,288 tokens/step, 2000 steps = 1.05B tokens
- Init seed: 1000 (same as exp-084, for comparability)
- Data seed: 2000 (same as exp-062 runs)
- Checkpoints at exp-062 checkpoint steps (including step 2000)

### Measurement

Same protocol as exp-062 and exp-084 (measure.py, exp-025/exp-059 pipeline):
- SEQ_LEN=512, N_INPUTS=50 random-token inputs, fp32
- FIT_LOW=3, FIT_HIGH=120, CENSUS_R2=0.85, CENSUS_DELTA=0.05
- Formation criterion: ≥ 10/48 conformal heads at step 2000

### Decision table

| C-generated result | Verdict |
|---|---|
| ≥ 10/48 conformal heads | H_transmission_yes: driver transmits with generated text |
| < 10/48 conformal heads | H_transmission_no: driver does not transmit; real reference matters |

## The escalation ladder (§6.3 from exp-062)

exp-062 → imprint/universality insufficient; reference or composition required  
exp-084 → hierarchy/composition insufficient; reference survives first kill test  
exp-085 → **does generation transmit the reference trace?**  

If H_transmission_no: the driver requires *actual* world-reference, not just its
statistical shadow. The next probe would examine finer-grained aspects of world-reference
(named entities vs. generic nouns, causal chains vs. random events, cross-document
co-reference vs. single-document text).

If H_transmission_yes: the conformal phase is induced by the distributional properties
of natural language, which a trained model can replicate. The next probe: is the driver
in the MI structure of the generated text, or in subtler higher-order properties?
